## Escenario del Proyecto

Escenario D – Estadísticas de Resultados Deportivos.

El proyecto analiza resultados de partidos de la Premier League para generar estadísticas básicas del torneo utilizando Python, GitHub, Jira y Google Colab.

---

## Objetivos

- Calcular partidos ganados por cada equipo
- Generar tabla de posiciones
- Calcular promedio de goles por partido
- Crear gráficos comparativos de rendimiento

---

## Dataset Utilizado

Se utilizarán datos públicos de la Premier League obtenidos desde:

https://www.football-data.co.uk/englandm.php

Los datos se almacenan dentro de la carpeta `/datos`.
